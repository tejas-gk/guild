## steps
```
- turn on xampp
- git clone this repo
- add envs
- cd server && composer install
- php artisan key:generate
- php artisan migrate:fresh --seed
- php artisan serve
- cd client && npm i
- npm run dev
- ./vendor/bin/pint
- npm run lint
```
